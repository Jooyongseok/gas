## 1. GasLeakage-MultiModal-MTF — 가스 누출 탐지 AI

### 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 목적 | 전자코(E-nose) + 열화상 카메라 데이터를 융합하여 가스 누출 탐지 및 분류 |
| 논문 기반 | Omneya Attallah, *Chemosensors* 2023 (DOI: 10.3390/chemosensors11070364) |
| 발표 | 한국정보과학회 학부생/주니어 논문 경진대회 31위 |
| 성능 | Intermediate Fusion: **98.47%** / Multitask Fusion: **99.25%** |
| 분류 클래스 | NoGas / Smoke / Perfume / Mixture (4종) |

### 핵심 아키텍처 상세

#### 데이터 입력 구조

```
[열화상 이미지 224×224 RGB]          [센서 히트맵 224×224 RGB]
        |                                       |
        |  7개 센서값 → 세로 스트라이프 변환       |
        |  (MQ2/MQ3/MQ5/MQ6/MQ7/MQ8/MQ135)     |
```

**센서 히트맵 변환 로직 (heat_transform.py)**
- 7개 MOS 센서값을 0~255로 정규화
- 224px 너비를 7등분 (각 32px) → 세로 스트라이프
- RGB 3채널 PNG로 저장 (카테고리별 1,600장 생성)

#### 모델 Forward Pass

```
열화상 이미지
    ├─ ResNet50   → apply_dwt → 1024차원
    ├─ Inception  → apply_dwt →  384차원
    └─ MobileNet  → apply_dwt →  640차원

센서 히트맵
    ├─ ResNet50   → apply_dwt → 1024차원
    ├─ Inception  → apply_dwt →  384차원
    └─ MobileNet  → apply_dwt →  640차원

→ torch.cat (6개 특징 벡터 연결) = 4096차원
→ apply_dct (DCT 주파수 변환)
→ BiLSTM (hidden=256, bidirectional) → Dropout(0.3)
→ Linear(512 → 4) → 최종 분류
```

#### 신호 처리 변환

| 변환 | 설명 | 목적 |
|------|------|------|
| DWT (Discrete Wavelet Transform) | Haar 웨이블릿, 각 CNN 출력에 적용 | 공간-스펙트럼-시간 표현 추출 |
| DCT (Discrete Cosine Transform) | 융합된 전체 특징에 적용 | 차원 축소 + 주파수 도메인 변환 |

## 2. etf-trading-project — AI ETF 자동 매매 시스템

### 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 목적 | AI 기반 ETF 종목 순위 예측 + 자동 매매 + 모니터링 대시보드 |
| 핵심 아이디어 | Daily Rolling(매일 포트폴리오 1% FIFO 교체)으로 패시브 ETF 대비 초과 수익 |

| 인프라 | Docker Compose + FastAPI + Next.js + MySQL + SQLite |
| 배포 환경 | ahnbi2.suwon.ac.kr (Nginx Reverse Proxy) |

### ETF 운용 핵심 로직 (Daily Rolling Cohort)

```
매일 거래일:
1. 포트폴리오에서 가장 오래된 매수 블록(1%) 전량 매도 (FIFO)
2. ML 서비스에서 당일 랭킹 상위 종목 조회
3. 해당 종목 신규 매수 (코호트 편입)
4. 상관계수 0.7 규제 모니터링 → 위반 시 벤치마크 추종 모드 전환
```

## 4. 발전 방향 및 추후 기획

---

### GasLeakage 프로젝트 발전 방향

#### 단기 개선 (1~3개월)

**① 실시간 추론 API 서버화**
- 현재: 로컬 스크립트 실행
- 개선: FastAPI 서버로 래핑 → HTTP 요청으로 실시간 가스 탐지
- 효과: 스마트 공장, IoT 게이트웨이 연동 가능

**② 모델 경량화**
- ResNet50 × 6개 = GPU 메모리 과부하
- MobileNet + EfficientNet-B0만 사용하는 경량 버전 분리
- 엣지 디바이스(라즈베리파이, Jetson Nano) 배포 가능

**③ 데이터 증강**
- 현재 1,600장/클래스 → 노이즈 주입, 센서값 스케일 변환으로 배수 증가
- 클래스 불균형 방지용 오버샘플링(SMOTE) 적용

#### 중기 개선 (3~6개월)

**④ 멀티모달 → 실제 IoT 파이프라인 구축**
```
[MOS 가스 센서] → [MCU/Raspberry Pi] → [실시간 스트림]
[열화상 카메라]  →         ↓
                  [Edge AI 추론 엔진]
                          ↓
                  [알람 시스템 / 대시보드]
```

**⑤ 센서 종류 확장**
- MQ 시리즈 7개 → 추가 화학 센서 (CO, NH3, H2S 등) 통합
- 음향 센서(누설음) 추가로 트리모달 융합 시도

**⑥ 설명 가능 AI (XAI) 적용**
- Grad-CAM으로 열화상 이미지의 어느 부위에서 가스 탐지되는지 시각화
- 어떤 센서가 분류에 기여했는지 SHAP 값으로 분석

#### 장기 비전 (6개월 이후)

**⑦ ETF 프로젝트와 연계: 산업 안전 데이터 플랫폼**
- 가스 누출 탐지 → 사고 이력 DB 구축
- 보험/리스크 평가 AI 모델로 확장
- 스마트 공장 ESG 지표에 연동

**⑧ 논문 후속 연구**
- Vision Transformer(ViT) 기반 열화상 특징 추출로 CNN 대체
- 시계열 센서 데이터에 Temporal Convolutional Network(TCN) 적용
- 논문 투고 (MDPI Sensors, IEEE Transactions on Industrial Electronics)

### ETF Trading 프로젝트 발전 방향

#### 단기 개선 (1~3개월)

**① 모델 고도화**
- 현재: LightGBM LambdaRank
- 추가: Transformer 기반 시계열 예측 (Temporal Fusion Transformer, PatchTST)

### 두 프로젝트 융합 기획: AI Safety-Finance Platform

> 두 프로젝트가 서로 다른 도메인이지만, **멀티모달 AI + 실시간 모니터링** 이라는 공통 기반 위에서 융합 가능

#### 융합 시나리오: 스마트 공장 ESG 투자 AI

```
[가스 누출 탐지 시스템]
    │ 산업 사고 데이터 / 안전 지표
    ▼
[ESG 스코어링 엔진]
    │ 기업 안전 등급화
    ▼
[ETF 종목 선정 AI]
    │ 안전 우수 기업 가중치 부여
    ▼
[ESG Active ETF 상품]
```

**기대 효과**
- 금융 + 산업 안전 데이터 결합 → 차별화된 ESG ETF
- "안전한 기업에 투자"라는 명확한 스토리텔링
- 국내 ESG 규제 강화 트렌드에 선제 대응
